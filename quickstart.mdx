<Steps>
  <Step title="Install the Quotient SDK">
    <CodeGroup>
      ```console pip
      pip install quotientai
      ```

      ```console npm
      npm install quotientai
      ```
    </CodeGroup>
  </Step>
  <Step title="Create an API Key">
    Next, create an API key to authenticate yourself. You can create an API key in your [settings](https://app.quotientai.co).
  </Step>
  <Step title="Initialize the Logger">
    Create a script to initialize the Quotient logger:

    <CodeGroup>
        ```python logging.py
        from quotientai import QuotientAI, DetectionType

        # initialize the quotient client
        quotient = QuotientAI()

        # initialize the logger
        quotient.logger.init(
            app_name="my-first-app",
            environment="dev",
            sample_rate=1.0,
            detections=[DetectionType.HALLUCINATION, DetectionType.DOCUMENT_RELEVANCY],
            detection_sample_rate=1.0
        )

        # log your first event
        response = quotient.log(
            user_query="How do I cook a goose?",
            model_output="The capital of France is Paris",
            documents=["Here is an excellent goose recipe..."]
        )

        print(response)
        ```

        ```typescript logging.ts
        import { QuotientAI, DetectionType } from 'quotientai';

        const quotient = new QuotientAI();

        quotient.logger.init({
            appName: "my-first-app",
            environment: "dev",
            sampleRate: 1.0,
            detections: [DetectionType.HALLUCINATION, DetectionType.DOCUMENT_RELEVANCY],
            detectionSampleRate: 1.0
        });

        const logId = await quotient.log({
            userQuery: "How do I cook a goose?",
            modelOutput: "The capital of France is Paris",
            documents: ["Here is an excellent goose recipe..."]
        });

        console.log(logId);
        ```

    </CodeGroup>
  </Step>
  <Step title="Run the Script">
    Using your API key, run the script to create your first log:
    <CodeGroup>
        ```console python
        QUOTIENT_API_KEY=your-api-key python logging.py
        ```

        ```console npx
        QUOTIENT_API_KEY=your-api-key npx ts-node logging.ts
        ```
    </CodeGroup>
  </Step>
  <Step title="View Your Logs">
    You just created your first log! Now you can go to the [Detections Dashboard](https://app.quotientai.co/detections) to see your logs and any detected hallucinations or document relevancy evaluations.
  </Step>
  <Step title="Analyze and Improve">
    Use the Detections Dashboard to:
    - Track hallucinations in your LLM responses
    - Evaluate document relevancy for your retrieval system
    - Analyze attribution between responses and source documents
    - Calculate hallucination rates across different environments
    - Optimize your prompts and retrieval system
  </Step>
</Steps>
